# TMS Module Catalog

Complete listing of all 100+ modules in Termux Master Suite.

## 🔐 Security & Penetration Testing

### Network Scanning
- **nmap** - Network discovery and security auditing
- **subfinder** - Fast subdomain discovery tool
- **amass** - In-depth attack surface mapping

### Web Application Testing
- **sqlmap** - Automatic SQL injection and database takeover
- **dirsearch** - Web path scanner
- **whatweb** - Web scanner
- **wapiti** - Web application vulnerability scanner
- **xsstrike** - XSS detection suite
- **redhawk** - Information gathering and vulnerability scanning

### Wireless Security
- **aircrack-ng** - WiFi security auditing suite
- **kismet** - Wireless network detector
- **reaver** - WPS brute force attack tool
- **wifite** - Automated wireless attack tool
- **bettercap** - Swiss Army knife for network attacks

### OSINT
- **sherlock** - Hunt down social media accounts by username
- **maigret** - Username OSINT tool
- **holehe** - Check if email is used on different sites
